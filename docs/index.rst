ausgsteckt implements a map showing "Buschenschenken" near you.
It uses multiple open data sources, like `OpenStreetMap <https://www.openstreetmap.org/>`_ and OpenGovData offered by Austrian Government for `Commune borders <https://www.data.gv.at/katalog/dataset/c33d36b0-f184-4f2a-89cc-839ca7fcf88a>`_.
