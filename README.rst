==========
ausgsteckt
==========

Requirements
============

* python >= 3.4 (<3.6 untested)
* postgresql with postgis
* see requirements.txt (dev libs like image stuff for easythumbnails, ...)

Quickstart
==========

* npm install
* pip install -r requirements.txt
* createdb ausgsteckt
* cd ausgsteckt
* python manage.py migrate
* python manage.py runserver
